# Teste-Pratico

**Projeto pratico desenvolvido para teste de conhecimento** <br />
**Ultilizado NodeJs, ReactJs, Typescript**
*Instalação* <br />

*SERVER* <br />
**No diretorio server :** <br />
***npm install*** <br />
***criar arquivo .env*** <br />
PORT=3001 <br />

*CLIENT* <br />
**Na pasta cliente** <br />
***npm install*** <br />
***npm run dev*** <br />
***acessar 127.0.0.1:3000*** <br />
